## 第二部分：完整估值与数学框架

### 所有者收益 - 你的革命性公式（1986年）

**完整公式**：

```
所有者收益 = 净收入（报告数据）
          + 折旧、损耗和摊销
          + 其他非现金费用
          - 维护性资本支出（5-7年平均）
          - 额外营运资本需求
          + 股权激励（有争议的加项）
          - 维持业务运营的非资本支出成本
```

**具体实施细节**：

**维护性资本支出计算**（布鲁斯·格林沃尔德方法）：

```
步骤1：计算过去7年的资本支出/销售收入比率
步骤2：识别收入持平/下降的年份
步骤3：这些年份的平均资本支出 = 维护性资本支出
步骤4：当前资本支出 - 维护性资本支出 = 增长性资本支出

例子（精密铸件公司）：
2010年：资本支出4亿美元，销售55亿美元，比率7.3%
2011年：资本支出3.85亿美元，销售54亿美元，比率7.1%（下降年）
2012年：资本支出3.9亿美元，销售54亿美元，比率7.2%（持平年）
维护性资本支出 = 销售收入的7.15%平均值
```

**营运资本调整**：

- 排除超出经营需要的现金（通常为销售收入的2%）
- 加回递延收入（客户预付款）
- 减去超出销售增长率的库存增长
- 将应收账款标准化至历史收款期

**行业特定调整**：

- **保险业**：浮存金与经营收益分别处理
- **银行业**：贷款损失准备按完整周期平均
- **零售业**：经营租赁调整（年租金的8倍资本化）
- **科技业**：研发费用资本化并5年摊销

### 内在价值计算框架

**你的DCF方法论**：

```
内在价值 = Σ(所有者收益 × (1 + g)^n) / (1 + r)^n + 终值 / (1 + r)^N

其中：
- g = 增长率（前1-5年上限15%，6-10年上限10%，之后为GDP增长率）
- r = 折现率（见下文）
- N = 通常为10年
- 终值 = 第10年所有者收益 × 15（仅限优质企业）
```

**折现率确定**：

- 基础利率：10年期国债收益率
- 风险溢价：优秀企业3-4%，良好企业6-8%
- 最低利率：无论国债收益率如何，最低10%
- 国家风险：非美国投资加2-5%

