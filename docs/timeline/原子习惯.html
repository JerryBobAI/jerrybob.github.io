习惯回路 The Habit Loop

提示 CUE

让它显而易见

Make It Obvious

  • 列出日常习惯: 觉察现有行为。
  • 习惯叠加: “在[旧习惯]后，执行[新习惯]”。
  • 环境设计: 优化空间，“一空间一用途”。

渴求 CRAVING

让它有吸引力

Make It Attractive

  • 诱惑捆绑: 喜欢的事 + 想养成的习惯。
  • 我们模仿: 加入积极的文化圈。
  • 激励仪式: 创造启动仪式进入状态。

反应 RESPONSE

让它简便易行

Make It Easy

  • 重复而非完美: 频率塑造习惯。
  • 减少阻力: 为未来扫清障碍。
  • 两分钟法则: 新习惯开始不超过2分钟。
  • 自动化: 尽可能自动化好习惯。

奖励 REWARD

让它令人愉悦

Make It Satisfying

  • 即时满足: 增加即时快乐。
  • 个人奖励计划: 可视化奖励。
  • 习惯追踪器: 衡量进步。
  • 古德哈特定律: 当衡量标准成为目标，不再是好标准。

如何戒除坏习惯 How to Break a Bad Habit?

让提示隐形

Make It Invisible

减少接触导致坏习惯的提示。

让它失去吸引力

Make It Unattractive

重塑心态，强调避免坏习惯的好处。

让它难以执行

Make It Difficult

增加执行坏习惯的步骤或阻力。

让它令人不快

Make It Unsatisfying

让坏习惯的代价公开且痛苦。

“这是一个持续的过程。没有终点线，小习惯会不断积累复利。让习惯变得愉快。带着厌倦感爱上它。”

This is a continuous process. No finish line. Small habits compound. Make habits enjoyable. Fall in love with boredom.
